# DX3D 프로젝트 컨텍스트

## 프로젝트 개요
- **목적**: DirectX12를 이용한 게임 엔진 개발
- **언어**: 한국어
- **개발 방식**: 직접 구현보다는 가이드 및 개념 위주의 학습

## 작업 규칙

### Q&A 문서화
사용자의 DirectX12 엔진 개발 관련 질문과 답변을 날짜별 마크다운 파일로 정리합니다.

**파일 형식**: `{yyyy-mm-dd}.md`
**저장 위치**: Docs 디렉토리

**문서 구조**:
```markdown
# DirectX12 엔진 개발 - YYYY-MM-DD

## 목차
- [간략한 주제1](#q1-질문-제목)
- [간략한 주제2](#q2-질문-제목)

---

## Q1: [질문 제목]
**질문:**
[질문 내용]

**답변:**
[답변 내용]

---

## Q2: [질문 제목]
...
```

**중요**:
- 새로운 Q&A를 추가할 때마다 문서 최상단의 목차에 해당 주제 링크를 즉시 추가
- 목차 항목은 `[간략한 주제 설명](#qN-질문-제목)` 형태로 작성
- 마크다운 앵커는 제목의 소문자 변환 + 공백을 하이픈으로 변환하여 자동 생성

### 작업 방식
1. 사용자가 DirectX12 관련 개념/가이드 질문을 함
2. 답변 제공
3. 해당 날짜의 마크다운 파일에 Q&A 형식으로 기록
4. 같은 날짜 내 추가 질문은 같은 파일에 순차적으로 추가
5. 날짜가 바뀌면 새로운 파일 생성

### 주의사항
- 코드 직접 구현 요청이 아닌 개념 설명 위주
- 답변은 명확하고 구조화된 형태로 제공
- DirectX12 API, 렌더링 파이프라인, 최적화, 멀티 스레딩 등 엔진 개발에 필요한 모든 주제 포함
